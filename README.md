# Tabungan Solvix Studio

Website pencatatan keuangan harian yang responsif dan interaktif untuk memantau pemasukkan, pengeluaran, target, hingga laporan. Dibangun dengan fokus pada pengalaman mobile karena mayoritas penggunaan berasal dari perangkat seluler.

## Keunggulan
- Mobile-first UI: tata letak ringkas dan nyaman dipakai di layar kecil.
- Multi tema kartun & standard: tampilan bisa diganti sesuai mood.
- History transaksi detail: data pemasukkan/pengeluaran langsung tampil di history.
- Report visual: grafik pemasukkan & pengeluaran bulanan/tahunan + export PDF/CSV.
- Interaksi kaya: validasi form, pop up konfirmasi, dan komponen UI yang konsisten.
- Fitur akun lengkap: sign in/sign up, profil, pengaturan, dan bahasa Indonesia/English.

## Fitur Utama
- CRUD Pemasukkan & Pengeluaran
- Ringkasan keuangan dan saldo otomatis
- Target/Dreams tracker
- Kalkulator
- Laporan (grafik & export)
- History transaksi dengan range tanggal

## Cara Menjalankan
Buka `index.html` di browser, atau jalankan server lokal sederhana (mis. Live Server).
